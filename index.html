Parte 01

HTML

HyperText Markup Language

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar de marcado utilizado para crear y estructurar el contenido de páginas web. No es un lenguaje de programación, sino un lenguaje de marcado que define la semántica y la estructura del contenido mediante el uso de etiquetas (tags).

Fue creado por Tim Berners-Lee en 1991 y es la base de toda página web. Los navegadores (Chrome, Firefox, Safari) interpretan el código HTML y lo convierten en la interfaz visual que los usuarios ven. Junto con CSS y JavaScript, forma la tríada fundamental del desarrollo web front-end.

  • Define la estructura del documento (encabezados, párrafos, listas…)
  • Establece la semántica del contenido (qué significan los elementos)
  • Permite hipervínculos entre documentos (la «HT» de HyperText)
  • Integra multimedia: imágenes, audio, video y formularios

Parte 02

CSS

Cascading Style Sheets

¿Qué es CSS?

CSS (Cascading Style Sheets / Hojas de Estilo en Cascada) es el lenguaje utilizado para describir la presentación de un documento HTML. Controla colores, tipografía, espaciado, posicionamiento, animaciones y cómo se adapta el diseño a distintos tamaños de pantalla.

El término «cascada» hace referencia al orden en que las reglas se aplican: cuando múltiples reglas afectan a un mismo elemento, el navegador determina cuál prevalece según la especificidad, el orden de aparición y la herencia.

  • Separa el contenido (HTML) de la presentación (CSS)
  • Permite reutilizar estilos en múltiples páginas
  • Controla responsividad y adaptación a diferentes dispositivos
  • Soporta animaciones y transiciones sin JavaScript
Parte 02 · A

Sintaxis CSS

Estructura de una regla CSS

Una regla CSS se compone de un selector y un bloque de declaraciones. Cada declaración contiene una propiedad y un valor separados por dos puntos, y las declaraciones se separan con punto y coma.

h1 {
color : #e63946 ;
font-size : 2rem ;
margin-bottom : 1rem ;
}
Selector Propiedad Valor
Componente Descripción Ejemplo
Selector Indica qué elemento(s) HTML serán afectados. h1, .clase, #id
Bloque { } Agrupa las declaraciones que se aplican al selector. { ... }
Propiedad La característica que se quiere modificar. color, font-size
Valor El valor asignado a la propiedad. #ff0000, 2rem
Parte 02 · B

Tipos de Selectores

1. Selectores Simples

Seleccionan elementos directamente por su tipo, clase, ID o de forma universal.

TipoDescripciónEjemplo de código
Elemento Selecciona todos los elementos del tipo indicado. p { color: gray; }
Clase (.) Selecciona todos los elementos con esa clase. .destacado { background: yellow; }
ID (#) Selecciona el único elemento con ese identificador. #header { height: 80px; }
Universal (*) Selecciona todos los elementos del documento. * { box-sizing: border-box; }

2. Selectores Combinadores

Combinan dos selectores para describir una relación de parentesco entre elementos.

TipoSímboloDescripciónEjemplo
Descendiente (espacio) Cualquier elemento descendiente, sin importar el nivel. div p { color: blue; }
Hijo directo > Solo el hijo directo (primer nivel). ul > li { list-style: none; }
Hermano adyacente + El elemento hermano inmediatamente siguiente. h2 + p { margin-top: 0; }
Hermano general ~ Todos los hermanos siguientes del mismo padre. h2 ~ p { color: gray; }

3. Selectores de Pseudoclase

Seleccionan elementos en un estado específico o según su posición.

PseudoclaseDescripciónEjemplo
:hover Cuando el cursor está sobre el elemento. a:hover { color: red; }
:focus Cuando el elemento tiene el foco (ej. campo activo). input:focus { border-color: blue; }
:nth-child(n) Selecciona el enésimo hijo de su padre. tr:nth-child(even) { background: #f0f0f0; }
:first-child Primer hijo de su elemento padre. li:first-child { font-weight: bold; }
:last-child Último hijo de su elemento padre. li:last-child { border: none; }
:not(selector) Selecciona todos los que NO coincidan con el selector dado. p:not(.especial) { color: gray; }
:checked Inputs de tipo checkbox o radio que estén marcados. input:checked + label { color: green; }

4. Selectores de Pseudoelemento

Permiten estilizar una parte específica de un elemento.

PseudoelementoDescripciónEjemplo
::before Inserta contenido generado antes del contenido del elemento. h2::before { content: "→ "; }
::after Inserta contenido generado después del contenido del elemento. .precio::after { content: " MXN"; }
::first-line Estiliza solo la primera línea de un bloque de texto. p::first-line { font-weight: bold; }
::first-letter Estiliza solo la primera letra (efecto capitular). p::first-letter { font-size: 2em; float: left; }
::selection Estiliza el texto seleccionado por el usuario. ::selection { background: #ffd700; }
::placeholder Estiliza el texto de marcador de posición en inputs. input::placeholder { color: #aaa; font-style: italic; }

5. Selectores de Atributo

Seleccionan elementos según la presencia o valor de un atributo HTML.

SelectorDescripciónEjemplo
[atributo] Elementos que poseen el atributo (independientemente del valor). [disabled] { opacity: 0.5; }
[atributo="valor"] Elementos cuyo atributo tiene exactamente ese valor. [type="submit"] { background: green; }
[atributo^="valor"] El valor del atributo empieza con la cadena indicada. [href^="https"] { color: green; }
[atributo$="valor"] El valor del atributo termina con la cadena indicada. [src$=".pdf"] { icon: pdf; }
[atributo*="valor"] El valor del atributo contiene la cadena indicada. [class*="btn"] { cursor: pointer; }
Parte 02 · C

Formas de Insertar CSS

1. CSS Externo

Se escribe en un archivo .css separado y se enlaza desde el HTML con la etiqueta <link> en el <head>. Es el método recomendado porque permite reutilizar estilos en múltiples páginas.

<!-- En el archivo HTML -->
<head>
  <link rel="stylesheet" href="estilos.css">
</head>

/* En el archivo estilos.css */
body {
  background-color: #f5f5f5;
  font-family: sans-serif;
}

2. CSS Interno (Embebido)

Se escribe dentro de una etiqueta <style> en el <head>. Útil para páginas pequeñas o estilos específicos de una sola página.

<head>
  <style>
    h1 {
      color: #333;
      text-align: center;
    }
    .contenedor {
      max-width: 1200px;
      margin: 0 auto;
    }
  </style>
</head>

3. CSS en Línea (Inline)

Se aplica directamente en el atributo style de la etiqueta HTML. Solo afecta a ese elemento específico. Se usa en casos muy puntuales; generalmente no recomendado porque mezcla contenido y presentación.

<p style="color: red; font-size: 18px; font-weight: bold;">
  Este párrafo tiene estilo en línea.
</p>

<button style="background: #007bff; color: white; padding: 8px 16px;">
  Botón estilizado
</button>
MétodoVentajaDesventajaCuándo usarlo
Externo Reutilizable, limpio, caché del navegador Requiere petición HTTP adicional Siempre (proyectos reales)
Interno Sin petición adicional No reutilizable entre páginas Prototipado rápido, email HTML
Inline Máxima especificidad Difícil de mantener, mezcla preocupaciones Casos muy excepcionales o JS dinámico

Temas Avanzados

Especificidad en CSS

La especificidad determina qué regla CSS «gana» cuando múltiples reglas apuntan al mismo elemento. Se calcula como un valor de 4 niveles: (a, b, c, d).

1,0,0,0
Inline style
style="..."
0,1,0,0
ID
#id
0,0,1,0
Clase / Pseudoclase / Atributo
.clase · :hover · [attr]
0,0,0,1
Elemento / Pseudoelemento
p · ::before

!important anula cualquier especificidad, pero su uso excesivo hace el código difícil de mantener.

Unidades de Medida

UnidadTipoDescripciónEjemplo
px Absoluta Píxeles. Medida fija, independiente del contexto. font-size: 16px;
em Relativa Relativa al font-size del elemento padre. margin: 1.5em;
rem Relativa Relativa al font-size del elemento raíz (html). Más predecible. padding: 1rem;
% Relativa Porcentaje respecto al elemento contenedor. width: 50%;
vw / vh Relativa 1% del ancho / alto del viewport. height: 100vh;
fr Flexible Fracción del espacio disponible (solo en Grid). grid-template-columns: 1fr 2fr;

Flexbox (Conceptos Básicos)

Flexbox (Flexible Box Layout) es un modelo de diseño unidimensional que distribuye el espacio entre ítems en un contenedor, ya sea en fila o columna.

/* Contenedor flex */
.contenedor {
  display: flex;
  flex-direction: row;          /* row | column | row-reverse | column-reverse */
  justify-content: space-between; /* alineación eje principal */
  align-items: center;           /* alineación eje cruzado */
  flex-wrap: wrap;              /* permitir salto de línea */
  gap: 16px;
}

/* Ítem flex */
.item {
  flex: 1;                       /* flex-grow flex-shrink flex-basis */
  align-self: flex-start;        /* sobreescribe align-items para este ítem */
}

📱 Media Queries (Responsividad)

Las media queries permiten aplicar estilos condicionales según las características del dispositivo: ancho de pantalla, orientación, resolución, etc. Son la base del diseño responsivo.

/* Estilos para pantallas menores a 768px (móvil) */
@media (max-width: 768px) {
  .contenedor {
    flex-direction: column;
    padding: 16px;
  }
  nav ul {
    display: none;
  }
}

/* Estilos para tablets (768px – 1024px) */
@media (min-width: 768px) and (max-width: 1024px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Preferencia de modo oscuro del sistema */
@media (prefers-color-scheme: dark) {
  body {
    background: #111;
    color: #eee;
  }
}

Transiciones y Animaciones

CSS permite crear efectos de movimiento sin JavaScript mediante transition (cambio suave entre estados) y animation (movimiento definido con @keyframes).

/* Transición */
.boton {
  background: #007bff;
  transition: background 0.3s ease, transform 0.2s ease;
}
.boton:hover {
  background: #0056b3;
  transform: scale(1.05);
}

/* Animación con @keyframes */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}
.elemento {
  animation: fadeIn 0.6s ease forwards;
}
